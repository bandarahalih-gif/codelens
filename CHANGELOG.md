# Changelog

All notable changes to CodeLens will be documented in this file.

## [0.4.0] - 2026-05-20
### Added
- Architecture diagram and screenshots in README
- Comprehensive API documentation
### Fixed
- Improved error handling for malformed code input
- Fixed token counting accuracy for large files

## [0.3.0] - 2026-05-14
### Added
- Docker multi-stage build support
- GitHub Actions CI pipeline
- Contributing guidelines
### Changed
- Improved test coverage to 85%

## [0.2.0] - 2026-05-08
### Added
- CI/CD REST API endpoint (POST /api/v1/review)
- Token usage tracking and budget management
- Settings page for API key configuration
### Fixed
- Database connection pooling issues

## [0.1.0] - 2026-05-01
### Added
- Initial release
- Multi-pass code analysis engine
- MiMo v2.5 integration for deep analysis
- Dashboard with review history
- Severity-based finding classification (Critical/High/Medium/Low)
- Support for Python, JavaScript, Go, Rust, Java
